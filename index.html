Cloud Security Documentation
Everything you need to know about Cloud Security

Introduction

Welcome to Cloud Security – the ultimate protection bot for Discord! Keep your community safe from threats, raids, and malicious activity with our advanced moderation, verification, and security features. Cloud Security ensures a secure and seamless experience for your server, so you can focus on what matters most – building your community.

Key Features

  • Advanced auto-moderation with customizable rules
  • Anti-raid protection with intelligent detection
  • Comprehensive logging system for all server activities
  • Smart announcer for welcome and leave messages
  • Situation analysis for security threats
  • Scam detection to protect your community
  • Ticket system for support requests

Quick Setup Guide
Get Cloud Security up and running in minutes

Step 1: Invite the Bot

Add Cloud Security to your server by clicking the "Add to Server" button and selecting your Discord server.

Add to Server

Step 2: Configure Permissions

Ensure Cloud Security has the necessary permissions to function properly. We recommend giving it administrator access for full functionality.

Step 3: Basic Configuration

Use the /setup command to configure basic settings for your server, including moderation levels and logging channels.

Security Settings

Configure security features to protect your server

Security Level
Choose the appropriate security level for your server
Low
Basic protection against common threats. Suitable for small, private servers.
Medium
Balanced protection with moderate restrictions. Recommended for most servers.
High
Enhanced protection with stricter moderation. Ideal for larger public servers.
Extreme
Maximum protection with very strict moderation. For servers at high risk of raids.
